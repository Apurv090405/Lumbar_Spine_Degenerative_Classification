## Introduction

**LumbarNet** is a deep learning-based diagnostic system designed to classify **degenerative changes in lumbar spine** MRI scans using **YOLOv11x**. This project tackles the limitations of traditional diagnostic methods by automating the classification process across **multi-modal medical imaging** (Sagittal T1, Sagittal T2/STIR, and Axial T2).
